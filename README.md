# IntiVision

🚧 **Work in Progress**

IntiVision is an Edge AI project focused on real-time hand gesture recognition and event processing.

The project combines Computer Vision, Deep Learning, and a high-performance Go backend to build a scalable AI-powered event pipeline.

## Planned Features

* Real-time hand gesture recognition
* Custom dataset collection pipeline
* Deep Learning model training and evaluation
* High-performance Go backend
* Event streaming architecture
* Docker-based deployment
* PostgreSQL integration
* REST API for AI inference

> **Status:** The project is currently under active development.
